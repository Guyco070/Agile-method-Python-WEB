Use guide:

Home page:

 •  For an unregistered user - register for the site by clicking on the "sign up" button.

 •  Connect to the site by clicking on the "login" button and entering login information for the user as entered in the registration.



Home page for current user:

 All users:

 •  Show My Projects - View the projects for selection and further execution of actions.


Manager:

 •  Open New Project - Enter the page to fill in project details and create a new project.



Project page:

 All users:

 •  Viewing project details.

 •  Kanban board - Displays a kanban board for all user stories in the project.

 •  Back to user page - Back to main page for current user.

 •  Send mail - Enter the page to fill in email details and send to the desired users according to the current project and user type.


Manager:

 •  Change project details - Enter the Change project details page - only details that are filled in will be changed.

 •   Add user story - Create and fill in new user story information.

  
  
  
  

   
   

Canban board page:
All users:

 •  View all user stories in the project to select in order to continue to the user story Page.

 •  A user story will be painted red according to the following conditions:

  
  
  

  

Manager:

 •  Change the status of the user story execution.


Programmer:

 •  Change the status of the user story execution.


Client:

 •  Rating update for user stories performed (Done status).



User story page:

 All users:

 •  View details of the user story, including start and end dates, requirements and more.


Manager:

 •  Edit user story - change / delete user story - only details that are filled in will be changed.
